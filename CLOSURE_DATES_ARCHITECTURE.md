## Data Structure

```typescript
// Configuration Structure
interface ClosureDate {
  name: string              // "Christmas Day"
  description?: string      // "TinkerTank is closed..."
  recurring: boolean        // true (annual) / false (one-time)
  month?: number           // 1-12 (for recurring)
  day?: number            // 1-31 (for recurring)
  specificDate?: Date     // Full date (for one-time)
  year?: number          // Optional year restriction
}

// Example: Recurring Date
{
  name: "Christmas Day",
  description: "TinkerTank is closed for Christmas",
  recurring: true,
  month: 12,
  day: 25
}

// Example: Specific Date
{
  name: "Facility Maintenance",
  description: "Annual maintenance day",
  recurring: false,
  specificDate: new Date('2025-07-15')
}
```
